Struct Round

pub struct Round { /* private fields */ }
Expand description

State containing all information related to the round state of the game. In the round stage, players each play a turn where they can draw cards, play cards and use their character ability. After every player has played a turn, players will be able to select characters again. If one player reached six or more assets during a round, the game will move to Results instead.
