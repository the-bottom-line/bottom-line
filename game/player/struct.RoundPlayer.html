Expand description

The player type that corresponds to the Round stage of the game. During the round stage, each player has selected a character.

Swaps a list of card indexes card_idxs with the deck. Each asset that is swapped is put back into the deck and each liability that is swapped is put back into the liability deck. If succesful, returns the total number of assets (left) and liabilties (right) that were removed from the player’s hand. After this action, the player is able to draw the total number of returned cards.
